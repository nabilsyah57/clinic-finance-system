# 📌 Project Overview

This project is a comprehensive Excel-based clinic financial and operational management system designed to streamline healthcare administration, patient transactions, doctor fee calculations, financial reporting, and clinic operational workflows.

The system was developed based on real operational needs inside a healthcare clinic environment and is actively used in daily clinic operations.

The goal of this project is to:
- Reduce manual calculation errors
- Improve operational efficiency
- Automate doctor fee calculations
- Simplify financial reconciliation
- Improve reporting accuracy
- Centralize clinic operational data

---

Features:

- Daily transaction grouping
- Automatic total fee calculation

Operational Benefits:

## 2. ⚡ Automated Daily Doctor Fee Input
Sheet: `Input Harian`

Features:
- Automatic patient counting by doctor
- Fee grouping by consultation amount
- Auto-generated daily fee recap
- Real-time synchronization from patient data

Calculation Logic:
- Uses COUNTIFS and SUMIFS formulas
- Automatically categorizes patient consultation fees
- Calculates daily totals per doctor
